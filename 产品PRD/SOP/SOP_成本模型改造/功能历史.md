# SOP 成本模型改造 - 功能历史

本文件记录 SOP 成本模型改造需求的所有迭代历史。

---

## 20260316

**迭代目的**：成本明细行表 metric_type 拆分为 metric_type + cost_sub_category，职责分离、便于扩展

**与上一轮差异**：
1. metric_type 仅表示计算方式（UNIT/FIXED/RATE），不再混入业务语义
2. 新增 cost_sub_category 存业务子类型（VIOLATION/HOLDING/CAPITAL_OCCUPATION/LATE_PENALTY/NON_DELIVERY）
3. 库存持有成本改为按单件计费（metric_type='UNIT'）

**主要变更**：

### 1. 表结构 sop_pla_finance_cost_line
- **metric_type**：枚举改为 `UNIT` / `FIXED` / `RATE`，表示计算方式（按件/按次/按率）
- **cost_sub_category**：新增字段，可空，枚举：`VIOLATION` / `HOLDING` / `CAPITAL_OCCUPATION` / `LATE_PENALTY` / `NON_DELIVERY`
- **唯一键 uk_cost_dim**：增加 `COALESCE(cost_sub_category, '')`

### 2. 成本映射关系
| cost_category | metric_type | cost_sub_category | 业务含义 |
|---------------|-------------|-------------------|----------|
| PRODUCTION | UNIT/FIXED | (空) | 单位/固定生产成本 |
| PURCHASE | UNIT/FIXED | (空) | 单位/固定采购成本 |
| TRANSPORT | UNIT/FIXED | (空) | 单位/固定运输成本 |
| INVENTORY | UNIT | VIOLATION | 单位最大库存违反成本 |
| INVENTORY | UNIT | HOLDING | 库存持有成本（按单件） |
| INVENTORY | RATE | CAPITAL_OCCUPATION | 资金占用成本率（年化） |
| PENALTY | UNIT | LATE_PENALTY | 单位晚交罚金 |
| PENALTY | UNIT | NON_DELIVERY | 单位欠交罚金 |

### 3. 导入模板
- 成本度量类型：枚举改为 单位成本/固定成本/按率计费（对应 UNIT/FIXED/RATE）
- 新增「成本子类」列：按需填写，生产/采购/运输成本可不填

### 4. 可扩展性
- 新增计算方式：扩展 metric_type
- 新增业务子类型：扩展 cost_sub_category
- 能耗示例：metric_type 用 UNIT/FIXED/RATE，cost_sub_category 新增 ENERGY_UNIT、ENERGY_FIXED、CARBON 等

## 20260314164439

**迭代目的**：新增库存资金占用成本、优化初始化交互、新增成本合计列

**与上一轮差异**：
1. 新增库存资金占用成本计算功能
2. 优化初始化按钮交互，增加两层确认弹窗
3. 各成本TAB新增成本合计列

**主要变更**：

### 1. 全局参数新增
- 新增参数：库存点物品的库存资金占用成本（启用/禁用）
- 新增参数：默认资金占用成本率（年化），百分比类型，默认值0%

### 2. 成本明细-库存成本扩展
- 新增成本度量类型：`CAPITAL_OCCUPATION`（资金占用成本率）
- 字段：资金占用成本率（年化），单位：%
- 表结构：`metric_type='CAPITAL_OCCUPATION'` 在 `cost_category='INVENTORY'` 下（*20260316 已调整为 metric_type='RATE' + cost_sub_category='CAPITAL_OCCUPATION'*）

### 3. 算法计算逻辑新增
- 新增第11条：CAPITAL_OCCUPATION_COST（库存资金占用成本）
- 计算公式：库存资金占用成本 = 平均库存价值 × 资金占用成本率（年化） × 时间系数
- 平均库存价值 = (期初库存 + 期末库存) / 2 × 单位价值
- 单位价值获取优先级：售价 > 单位生产成本 > 单位采购成本
- 时间系数：月=1/12，周=1/52，日=1/365

### 4. 初始化按钮交互优化
**适用范围**：售价明细、成本明细各TAB的初始化按钮

**交互流程**：
- 场景A：用户勾选了部分数据 → 直接弹出"确认初始化"弹窗（第二层）
- 场景B：用户未勾选任何数据 → 先弹出"全量初始化警告"弹窗（第一层）→ 点击"继续"后弹出"确认初始化"弹窗（第二层）

**弹窗设计**：
- 第一层弹窗：全量初始化警告，提示将对当前TAB所有数据执行初始化
- 第二层弹窗：确认初始化，提示初始化范围和建议导出备份

### 5. 成本合计列新增
各TAB新增成本合计列（自动计算、只读、灰色背景）：
- 生产成本TAB：生产成本合计 = 单位生产成本 + 固定生产成本
- 采购成本TAB：采购成本合计 = 单位采购成本 + 固定采购成本
- 运输成本TAB：运输成本合计 = 单位运输成本 + 固定运输成本
- 库存成本TAB：库存成本合计 = 单位最大库存违反成本 + 库存持有成本（不含资金占用成本率）
- 违约罚金TAB：违约罚金合计 = 单位晚交罚金 + 单位欠交罚金

**重要说明**：合计列仅用于表格展示，算法计算"单位价值"时仍然只使用"单位成本"（不含固定成本）

### 6. 数据校验逻辑更新
- 成本明细-库存成本校验新增：若全局参数"库存点物品的库存资金占用成本=启用"，检查资金占用成本率（年化）是否有空值

### 7. 原型图更新
- 成本明细-库存成本TAB新增"资金占用成本率（年化）"列
- 各成本TAB新增"成本合计"列
- 初始化按钮增加两层确认弹窗交互
