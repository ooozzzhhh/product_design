# 功能历史

本文件记录运行监控中心原型的迭代历史。

## 20260310103000 (2026-03-10 10:30:00)

**本轮目的**: 根据PRD 2.1.6-2.1.9新增的技术规范,在现有React + TypeScript项目基础上进行功能扩展

**新增功能**:
1. 并发控制规范 (PRD 2.1.6)
   - 支持PARALLEL/EXCLUSIVE/QUEUEABLE/LIMITED_PARALLEL四种并发模式
   - 为所有功能点配置添加并发控制配置
   - 新增排队状态展示(排队位置、预计等待时间)

2. 实时推送机制 (PRD 2.1.7)
   - 支持WebSocket实时推送(推荐实现,后续迭代)
   - 支持轮询降级(当前实现)
   - 支持断线重连与消息补发

3. 终止与异常跳过 (PRD 2.1.8)
   - 支持任务终止(二次确认对话框)
   - 支持节点级可中断控制(可中断/不可中断节点)
   - 支持异常跳过框架(可跳过/不可跳过节点)
   - 新增节点详情展示(业务指标、警告信息、错误信息)

4. 性能监控 (PRD 2.1.9)
   - 支持节点级业务指标记录
   - 支持性能监控接口(JVM、线程、数据库连接池)
   - 新增性能详情弹窗(查看性能指标)

**修改文件**:
- `types.ts`: 新增并发控制、排队状态、性能监控、节点详情等类型定义
- `constants.ts`: 为所有功能点配置添加并发控制配置
- `mockData.ts`: 新增排队状态和节点详情的模拟数据
- `components/RunProgressModal.tsx`: 新增终止确认对话框和性能详情弹窗
- `components/FunctionDetailView.tsx`: 新增排队状态展示和异常跳过节点展示

**与上一轮差异**: 本轮为首次迭代,无上一轮对比

**详细说明**: 见 `20260310103000/修改说明.md`
