# 智能MRP系统前端设计规范（AI提示词）

## 📋 设计概述

这是一套面向企业级后台管理系统的现代前端设计规范，强调**专业、清晰、高效**的视觉体验。整体风格偏向**商务科技感**，使用大圆角、柔和阴影和细腻的交互反馈。

---

## 🎨 颜色系统

### 主色调
- **主品牌色（深蓝紫）**: `#1a1f71` / `indigo-900`
  - 用于：主要按钮、Logo背景、重要强调元素
  - 悬停色：`#121650`
  
- **辅助品牌色（靛蓝）**: `#4f46e5` / `indigo-600`
  - 用于：链接、选中状态、进度条、图标背景
  - 浅色背景：`#eef2ff` / `indigo-50`

### 功能色
- **成功色**: `#10b981` / `emerald-500`
  - 浅色背景：`#ecfdf5` / `emerald-50`
  - 用于：成功状态、完成标记、正向反馈

- **错误色**: `#ef4444` / `rose-500`
  - 浅色背景：`#fef2f2` / `rose-50`
  - 用于：错误提示、失败状态、警告标记

- **警告色**: `#f59e0b` / `amber-500`
  - 浅色背景：`#fffbeb` / `amber-50`
  - 用于：警告提示、待处理状态

### 中性色（Slate系列）
- **背景色**:
  - 主背景：`#f3f6fb`（浅蓝灰，带轻微蓝色调）
  - 卡片背景：`white`
  - 输入框背景：`#f3f6fb` / `slate-50`
  
- **文字色**:
  - 主标题：`slate-900`（深灰黑）
  - 正文：`slate-700` / `slate-600`
  - 次要文字：`slate-500` / `slate-400`
  - 占位符：`slate-300`

- **边框色**:
  - 默认边框：`slate-100` / `gray-100`
  - 悬停边框：`slate-200`

### 阴影系统
- **卡片阴影**: `shadow-sm`（轻微阴影）
- **按钮阴影**: `shadow-lg shadow-indigo-100`（带颜色阴影）
- **模态框阴影**: `shadow-2xl`（大阴影）
- **悬停阴影**: `hover:shadow-md`

## 🔤 字体系统

### 字重层级
- **超粗体（标题）**: `font-black`（900）
  - 用于：页面主标题、重要数字、按钮文字
  
- **粗体（强调）**: `font-bold`（700）
  - 用于：副标题、标签、重要信息
  
- **半粗体（正文）**: `font-semibold`（600）
  - 用于：表格表头、次要标题
  
- **中等（常规）**: `font-medium`（500）
  - 用于：正文、描述文字

### 字号层级
- **超大标题**: `text-3xl`（30px）
- **大标题**: `text-2xl`（24px）
- **标题**: `text-xl`（20px）
- **正文**: `text-sm`（14px）、`text-base`（16px）
- **小字**: `text-xs`（12px）
- **极小字**: `text-[10px]`、`text-[11px]`（用于标签、元信息）

### 文字样式
- **大写字母**: `uppercase` + `tracking-widest`（用于标签、状态）
- **等宽字体**: `font-mono`（用于代码、订单号）
- **行高**: 默认行高，重要文本使用 `leading-tight`

---

## 📐 布局系统

### 间距系统
- **大间距**: `space-y-8`、`p-8`（32px）- 用于页面区块
- **中间距**: `space-y-6`、`p-6`（24px）- 用于组件内部
- **小间距**: `space-y-4`、`p-4`（16px）- 用于元素组
- **极小间距**: `gap-2`、`gap-3`（8px/12px）- 用于图标与文字

### 圆角系统
- **超大圆角**: `rounded-[2.5rem]`（40px）- 用于卡片、模态框
- **大圆角**: `rounded-2xl`（16px）- 用于按钮、输入框
- **中圆角**: `rounded-xl`（12px）- 用于标签、小卡片
- **小圆角**: `rounded-lg`（8px）- 用于徽章、小元素
- **圆形**: `rounded-full` - 用于头像、状态点

### 容器宽度
- **侧边栏**: `w-64`（256px）
- **主内容区**: `max-w-7xl`（1280px）居中
- **模态框**: `max-w-lg`（512px）、`max-w-xl`（576px）、`max-w-2xl`（672px）

### 布局模式
- **侧边栏布局**: 固定左侧导航 + 右侧主内容区
- **网格布局**: `grid grid-cols-12`、`grid-cols-4` 等
- **弹性布局**: `flex` + `flex-1` 用于自适应

## ✨ 动效系统

### 过渡效果
- **通用过渡**: `transition-all`（所有属性过渡）
- **过渡时长**: 默认 `duration-300`（300ms）
- **缓动函数**: 默认 `ease-out`

### 交互反馈
- **悬停效果**:
  - 背景色变化：`hover:bg-indigo-50`
  - 阴影增强：`hover:shadow-md`
  - 文字颜色：`hover:text-indigo-600`
  
- **点击反馈**:
  - 缩放：`active:scale-95`（缩小至95%）
  - 背景加深：`hover:bg-[#121650]`

### 动画类型
- **脉冲动画**: `animate-pulse` - 用于加载、警告状态
- **旋转动画**: `animate-spin` - 用于加载图标
- **淡入动画**: `animate-in fade-in duration-500` - 用于页面切换
- **缩放进入**: `animate-in zoom-in-95 duration-300` - 用于模态框
- **滑入动画**: `animate-in slide-in-from-top-4` - 用于下拉菜单

### 进度动画
- **进度条**: 使用 `transition-all duration-500 ease-out`
- **进度条内部光效**: `bg-white/20 animate-pulse`（白色半透明脉冲）

## 🎯 组件规范

### 按钮（Button）

#### 主要按钮
```css
- 背景：bg-[#1a1f71]
- 文字：text-white
- 圆角：rounded-2xl
- 内边距：px-8 py-3（或 h-12 px-6）
- 字重：font-black
- 字号：text-sm
- 阴影：shadow-lg shadow-indigo-100
- 悬停：hover:bg-[#121650]
- 点击：active:scale-95
- 图标：flex items-center gap-2
```

#### 次要按钮
```css
- 背景：bg-white
- 边框：border-2 border-slate-200（或 border-amber-200、border-rose-200）
- 文字：text-slate-500（或 text-amber-600、text-rose-500）
- 悬停：hover:bg-slate-50（或对应浅色背景）
```

#### 文本按钮
```css
- 背景：bg-[#f3f6fb]
- 文字：text-[#1a1f71]
- 悬停：hover:bg-[#e2e8f0]
```

### 输入框（Input）

```css
- 背景：bg-[#f3f6fb] 或 bg-slate-50
- 边框：border-none（或 border-2 border-slate-100）
- 圆角：rounded-2xl
- 内边距：pl-11 pr-4 py-3（带图标时左侧留空）
- 字号：text-sm
- 字重：font-bold
- 聚焦：focus:ring-2 focus:ring-indigo-500/20 focus:border-indigo-500
- 图标位置：absolute left-4 top-1/2 -translate-y-1/2
```

### 卡片（Card）

```css
- 背景：bg-white
- 边框：border border-gray-100
- 圆角：rounded-[2.5rem]（大卡片）或 rounded-xl（小卡片）
- 阴影：shadow-sm
- 悬停：hover:shadow-md transition-all
- 内边距：p-8（大卡片）或 p-4（小卡片）
```

### 表格（Table）

```css
- 容器：overflow-hidden border border-slate-200 rounded-xl bg-white shadow-sm
- 表头：bg-slate-50/80
- 表头文字：font-semibold text-slate-600 uppercase tracking-wider text-[11px]
- 表体：bg-white divide-y divide-slate-100
- 行悬停：hover:bg-slate-50/50 transition-all duration-200
- 单元格：px-4 py-3 或 px-4 py-4
```

### 模态框（Modal）

```css
- 遮罩：bg-slate-900/40 backdrop-blur-sm
- 容器：bg-white rounded-[2.5rem] shadow-2xl
- 动画：animate-in zoom-in-95 duration-300
- 头部：p-8 border-b border-slate-100 bg-slate-50/50
- 内容区：p-8 space-y-8 overflow-y-auto
- 底部：p-8 bg-slate-50 border-t border-slate-100
```

### 标签/徽章（Badge）

```css
- 成功：bg-emerald-50 text-emerald-700 border-emerald-100
- 警告：bg-amber-50 text-amber-700 border-amber-100
- 默认：bg-slate-50 text-slate-600 border-slate-100
- 圆角：rounded-md 或 rounded-lg
- 内边距：px-2.5 py-1
- 字号：text-[11px]
- 字重：font-bold uppercase tracking-wider
```

### 进度条（Progress）

```css
- 容器：h-3 w-full bg-slate-100 rounded-full overflow-hidden p-0.5 border border-slate-50
- 进度条：h-full rounded-full transition-all duration-500 ease-out
- 成功色：bg-[#10b981]
- 进行中：bg-[#4f46e5] + bg-white/20 animate-pulse
- 失败色：bg-rose-500
```

### 状态指示器

```css
- 成功点：w-2.5 h-2.5 rounded-full bg-[#10b981]
- 待处理点：w-2.5 h-2.5 rounded-full bg-[#94a3b8]
- 错误点：w-2.5 h-2.5 rounded-full bg-[#ef4444] + animate-pulse
- 图标容器：w-16 h-16 rounded-2xl flex items-center justify-center text-white text-2xl shadow-lg
```

## 🎭 图标系统

### 图标库
- **Font Awesome 6.4.0**（通过CDN引入）
- 使用 `<i className="fa-solid fa-*"></i>` 格式

### 常用图标
- **导航**: `fa-table-list`、`fa-square-poll-vertical`
- **操作**: `fa-bolt-lightning`、`fa-sliders`、`fa-hand`、`fa-bug`
- **状态**: `fa-check`、`fa-xmark`、`fa-loader`、`fa-triangle-exclamation`
- **功能**: `fa-magnifying-glass`、`fa-calendar-day`、`fa-brain`、`fa-code`

### 图标使用规范
- **尺寸**: `text-lg`（18px）、`text-xl`（20px）、`text-2xl`（24px）
- **颜色**: 跟随上下文，通常与文字颜色一致
- **间距**: 与文字使用 `gap-2` 或 `gap-3`
- **旋转**: 加载图标使用 `animate-spin`

## 🎪 特殊效果

### 背景模糊
- **模态框遮罩**: `backdrop-blur-sm`

### 渐变效果
- **进度条光效**: `bg-white/20 animate-pulse`（白色半透明脉冲）

### 边框高亮
- **聚焦状态**: `focus:ring-2 focus:ring-indigo-500/20`

### 状态动画
- **成功完成**: 绿色背景 + 白色对勾图标
- **进行中**: 蓝色背景 + 旋转加载图标 + 脉冲动画
- **失败/错误**: 红色背景 + 警告图标 + 脉冲动画

## 📱 响应式设计

### 断点策略
- **移动端**: 默认样式（< 1024px）
- **桌面端**: `lg:` 前缀（≥ 1024px）
- **网格响应**: `grid-cols-1 lg:grid-cols-12`

### 适配要点
- 侧边栏在移动端可折叠
- 表格在小屏幕横向滚动
- 模态框在小屏幕全屏显示

## 🎨 视觉层次

### 层级关系
1. **最高层（模态框）**: `z-[100]`、`z-50`、`z-[60]`
2. **导航层（侧边栏）**: `z-20`
3. **内容层**: 默认 `z-0`
4. **下拉菜单**: `z-20`

### 视觉权重
- **最重要**: 深色背景 + 白色文字 + 大阴影
- **重要**: 品牌色背景 + 白色文字 + 中等阴影
- **次要**: 白色背景 + 品牌色文字 + 边框
- **最次要**: 浅色背景 + 灰色文字

---

## 💡 设计原则总结

1. **大圆角设计**: 统一使用 `rounded-2xl` 及以上，营造现代感
2. **柔和阴影**: 使用带颜色的阴影（如 `shadow-indigo-100`）增加层次
3. **细腻交互**: 所有可交互元素都有悬停和点击反馈
4. **清晰层次**: 通过颜色、字重、间距建立清晰的视觉层次
5. **专业配色**: 深蓝紫主色 + 功能色 + 中性色，保持商务感
6. **动画适度**: 使用淡入、缩放、脉冲等动画，不过度使用
7. **图标辅助**: 重要操作和状态都配有图标，提升可读性

## 📝 使用示例

### 示例1：创建一个主要按钮
```tsx
<button className="h-12 px-8 bg-[#1a1f71] text-white rounded-2xl text-sm font-black hover:bg-[#121650] transition-all active:scale-95 flex items-center gap-2 shadow-lg shadow-indigo-100">
  <i className="fa-solid fa-bolt-lightning"></i>
  运行 MRP
</button>
```

### 示例2：创建一个输入框
```tsx
<div className="relative">
  <i className="fa-solid fa-magnifying-glass absolute left-4 top-1/2 -translate-y-1/2 text-slate-400"></i>
  <input 
    type="text" 
    className="w-full pl-11 pr-4 py-3 rounded-2xl border-none bg-[#f3f6fb] text-sm font-bold focus:ring-2 focus:ring-indigo-500/20"
    placeholder="搜索..."
  />
</div>
```

### 示例3：创建一个卡片
```tsx
<div className="bg-white border border-gray-100 rounded-[2.5rem] p-8 shadow-sm hover:shadow-md transition-all">
  <h3 className="text-xl font-black text-slate-900 mb-4">标题</h3>
  <p className="text-slate-600 font-medium">内容...</p>
</div>
```
