## 十三、测试执行说明

1. **执行顺序**：建议先完成「页面与导航」「搜索」「保存/重置」等 P0，再按分类（时间→排程→数据有效期→数据类型→界面→财务）逐项执行参数修改与帮助。
2. **环境**：使用支持 Vue 2 + Element UI 的浏览器打开原型；若已对接后端，需在测试环境验证保存与重置接口。
3. **数据**：重置后以原型中的 `originalParams` 或接口返回的默认值为基准，校验重置是否一致。
4. **回归**：全局参数为多模块共用，参数项或分类变更后建议回归本用例集及依赖全局参数的页面（如财务管理初始化）。
